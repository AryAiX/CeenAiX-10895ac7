# 7. Glossary

| Term | Definition |
|---|---|
| CeenAiX | The AI-native national healthcare intelligence platform developed by AryAiX |
| PHR | Personal Health Record — a longitudinal patient-owned health record maintained within CeenAiX |
| AI Health Agent | CeenAiX's conversational AI model powering consultations, analysis, and clinical assistance |
| SOAP Note | Structured clinical documentation format: Subjective, Objective, Assessment, Plan |
| DHA | Dubai Health Authority — primary healthcare regulatory body for the Emirate of Dubai |
| RBAC | Role-Based Access Control — security model restricting system access based on user role |
| Emirates ID | UAE national identity document used for patient identity verification and emergency access |
| Pre-Visit Assessment | AI-conducted structured health questionnaire completed by the patient before an appointment |
| CME | Continuing Medical Education — ongoing professional development for licensed healthcare providers |
| OCR | Optical Character Recognition — AI technology used to read text from uploaded photos and documents |
| ARR | Annual Recurring Revenue — the normalized annual value of recurring platform subscriptions |
| GCC | Gulf Cooperation Council — regional bloc including UAE, Saudi Arabia, Kuwait, Qatar, Bahrain, and Oman |
| RLS | Row-Level Security — PostgreSQL feature that restricts data access per row based on user identity |
| FHIR | Fast Healthcare Interoperability Resources — HL7 standard for exchanging healthcare information electronically |
| SFU | Selective Forwarding Unit — WebRTC server architecture for efficient multi-party video routing |
| PWA | Progressive Web App — web application installable on mobile devices with offline capabilities |
| JWT | JSON Web Token — compact token format used for authentication and authorization |
| SSE | Server-Sent Events — protocol for streaming server-to-client updates over HTTP |
| ICD-10 | International Classification of Diseases, 10th Revision — WHO standard for medical diagnosis coding |
| CPT | Current Procedural Terminology — standard coding system for medical procedures and services |
